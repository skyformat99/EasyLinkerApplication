# 内部的三种消息格式
## 数据消息
```
{
    "data":{
        "data":{
            "V1":"1",
            "V2":"2"
        },
        "persistent":"true",
        "info":"V"
    },
    "type":"data"
}
```
## 回显消息
### Topic:/system/echo
```
{
    "data":{
        "data":{
            "cmd":"ls"
        }
    },
    "type":"cmd"
}
```
## 特殊命令消息
### Topic:/system/cmd
```
{
    "data":{
        "data":{
            "echo":"echo"
        }
    },
    "type":"echo"
}
```
